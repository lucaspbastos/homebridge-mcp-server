# Homebridge MCP Server

Experimental MCP server for interfacing with a Homebridge installation.

## Install Node & Dependencies
```mise install```
- Installs Node 24, latest pnpm

```pnpm install```
- Installs Node depencencies

## Build & Run

### Local
Build:
```pnpm build```
- Builds TypeScript project

Run:
```pnpm run```
